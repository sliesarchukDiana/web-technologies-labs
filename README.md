# Web Development Laboratory Works 💻

## 📌 Overview
This repository contains the source code and project files for my laboratory assignments in Web Development. The exercises are divided into two main sections: structural/styling tasks using HTML and CSS, and interactive scripting tasks using JavaScript.

## 🛠️ Technologies Used
* **HTML5 & CSS3:** Semantic structure, web forms, responsive layouts, and visual styling.
* **JavaScript (Vanilla JS):** DOM manipulation, event handling, and client-side logic.

## 📂 Repository Structure

The repository is organized into two primary directories based on the core technologies used:

### 🎨 HTML & CSS Labs (`/lab-HTML_CSS`)
Contains assignments focused on markup and styling.
* 📁 `lab1/` - *Basic HTML tags]*
* 📁 `lab2/` - *Text formatting and lists]*
* 📁 `lab3/` - *Tables and links]*
* 📁 `lab4/` - *HTML Forms]*
* 📁 `lab5/` - *Intro to CSS styling]*
* 📁 `lab6/` - *CSS Box Model]*
* 📁 `lab7/` - *Flexbox layouts]*
* 📁 `lab8/` - *CSS Grid layouts]*
* 📁 `lab9/` - *Responsive design and media queries]*

### ⚡ JavaScript Labs (`/lab-JS`)
Contains assignments focused on interactivity and scripting.
* 📁 `lab-1/` - *Basic syntax and variables]*
* 📁 `lab-2/` - *Functions and control flow]*
* 📁 `lab-3/` - *Arrays and Objects*
* 📁 `lab-4/` - *Arrays and Objects*
* 📁 `lab-5/` - *setTimeout та setInterval, Map/Set. WeakMap/WeakSet*
